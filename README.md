# Challenge Conversor de Monedas

Conversor de monedas de USD a distintos tipos de monedas.

## 🔨 Funcionalidades del proyecto

- Elegir el tipo de conversion que quieres realizar.
- Ingresar el monto a convertir.
- Se realizara la conversion mostrando el resultado.

## ✔️ Técnicas y tecnologías utilizadas

- `Eleccion tipo de conversion`: Se elige el tipo de conversion a traves de un menu.
- `Ingresar monto`: A traves de la consola se ingresa el monto que se desea convertir.
- `Conversion`: A traves de una funcion se logra generar la conversion del tipo de cambio y mostrar el resultado.
